## About control flow statement
- >Control blocks are used to show, hide and repeat the elements.

1. Use of **@if** statement:
```
@if(condition) {
    HTMl-Tag to be shown/hide
}
```

2. Use of **@if-else** statement:
```
@if (condition==true) {
    HTML tags
} @else {
    <!-- false condition -->
    HTML tags
}
```

3. Use of **if-else-if** statement:
```
@if(conditon1) {
    <!-- tags to be rendered -->
} @else if(condition2) {
    <!-- tags to be rendered -->
} @else(conditonLeft) {
    <!-- tags to be rendered -->
}
```
